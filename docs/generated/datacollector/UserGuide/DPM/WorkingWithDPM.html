Working with Control Hub

Before you can work with StreamSets Control Hub, you must have an Control Hub organization and user account.

To work with Control Hub, you complete the following tasks within the Data Collector UI:

  • Register Data Collector to work with Control Hub.
  • Configure pipelines to aggregate statistics.
  • Manage how pipelines work with Control Hub.
  • Optionally, modify Control Hub configuration properties.
  • Optionally, unregister Data Collector from working with Control Hub.
